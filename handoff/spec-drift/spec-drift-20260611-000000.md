### 2.2 drift 분류

**일치(의미 drift 없음)**

이번 v0.10 변경은 **파일명 `통합본_v0_9.md` 유지(DEC-1=(c))** — 형제 repo에서 경로로 참조하는 `../monitoring-meta/docs/통합본_v0_9.md`는 여전히 유효한 경로이므로 **파일 참조 경로 drift 없음**.

**기준 문서가 더 새로움 — 버전 호칭 한정**

"통합본 v0.9"라는 호칭은 기준 문서 내부에서 v0.10으로 갱신됐으나, 형제 repo의 에이전트 파일들은 여전히 "v0.9"로 인용한다. 이 항목의 영향 범위:

- **기능 동작 drift**: 없음. 내용·결정이 0 변경인 표기 전용 릴리스이므로 "v0.9"를 "v0.10"으로 읽어도 가리키는 spec 내용은 동일하다.
- **agent 파일의 버전 호칭 정합**: 에이전트 파일에 "통합본 v0.9"로 표기되어 있어도 경로(`../monitoring-meta/docs/통합본_v0_9.md`)는 정확하므로 운영상 오동작 위험 없음.
- **결론**: 표기 전용 변경이므로 에이전트 파일의 버전 호칭 업데이트는 선택적. 즉각 동기화 없이도 spec 운영에 지장 없음.

**`[Open]` 마커 인용 — drift 없음**

형제 repo 에이전트 파일에서 통합본의 `[Open]` 또는 `[Open question]` 마커를 직접 인용한 사례 없음.

**β/γ 의미 병기 — drift 없음**

`hub/.claude/agents/reviewer.md`가 β 구조를 직접 설명(`메인 BE = ... / 분리 대상 = ...`)하고 있어 v0.10에서 β 첫 등장부 병기가 추가됐더라도 **reviewer의 설명 내용이 기준 문서와 동일** — 의미 drift 없음.
